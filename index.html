Study Flashcard

JavaScript (JS)

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

A special symbol used to perform operations on operands (values and variables)

Variables

Used to store information to be referenced and used by programs.

Variable naming conventions

A set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities.

Functions

A set of instructions bundled together to achieve a specific outcome.

Methods

A kind of function that is called on something.

Arguments

A way to provide more information to a function; variabls used only in that specific function.

Parameters

A special kind of variable used in a function to refer to one of the pieces of data provided as input to the function.

Return

Ends function execution and specifies a value to be returned to the function caller.

Chaining methods

A technique that is used for making multiple method calls on the same object-oriented programming languages.

Strings

A data type used to represent text rather than numbers.

Booleans

A data type with the possible values: true or false.

Undefined

A condition where an expression does not have a correct value, although it is syntactically correct.

NaN

Stands for Not a Number. A numeric data type that can be interpreted as a value that is undefined or unrepresentable.

Null

A built-in constant that has a value of zero.

Alert

A type of "pop-up" message to users of a script.

Comments

A programmer-readable explanation or annotation in the source code.

jQuery

A JavaScript library designed to bundle common tasks of JavaScript code into various methods.

Attributes

A specification that defines a property of an object, element, or file.
